# yze-go-ctxfirst

A [`yze`](https://github.com/gomatic/yze) analyzer (group `go`, category `patterns`) enforcing the gomatic Go idiom that a `context.Context` parameter is always the **first** parameter of a function or method.

- **Rule:** `yze/go/ctxfirst`
- **Binary:** `cmd/yze-go-ctxfirst` runs it standalone.

Built on the [`go-yze`](https://github.com/gomatic/go-yze) framework.
